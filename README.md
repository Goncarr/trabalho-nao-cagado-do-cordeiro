Descrição:
-A identidade visual da landing page realizada baseia-se na simplicidade e legibilidade. Utiliza na maioria das circunstâncias cores simples com contraste significativo entre elementos e texto em tamanhos legíveis e com conteúdo de facil leitura.


Evolução das várias fases:
-Ambos o wireframe e o mockup foram realizados no figma, com o primeiro destes sendo basicamente apenas uma versão crude deste, a qual foi usada como base para a realização do mockup. O design do mockup segue fielmente o que foi apresentado no site, apresentando apenas eventuais diferenças imperceptíveis nas imagens.


Acessibilidade e legibilidade:
-Durante a realização deste projeto, teve-se o cuidado de usar fontes e elementos com tamanhos adequados ao seu dispositivo, bem como tornar responsivo para outras plataformas, sendo funcional em mobile, tablet e desktop.
